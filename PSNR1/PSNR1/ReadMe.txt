========================================================================
    CONSOLE APPLICATION : PSNR1 Project Overview
========================================================================

This application calculates the PSNR for two YUV format files.

The application takes 5 parameters, e.g
PSNR1  "C:\\Test_768x432_yuv420p.yuv" "C:\\Ref_768x432_yuv420p.yuv" 768 432 420

Parameter 1: first YUV file path name;
Parameter 2: second YUV file path name;
Parameter 3: YUV frame width;
Parameter 4: YUV frame height;
Parameter 5: YUV format. 3 YUV formats are supported: 444, 422, 420;
